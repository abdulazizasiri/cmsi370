Welcome to my Role Playing Game!

Game Description

The game allows you to do each of the following things.

  • Display a list of current characters
  • Spawn/create a new character
  • View a character
  • Modify a character
  • Delete a character
  • Spawn a random item

More Tips and Tricks

Display a list of current characters

In order to display the list of current users, you can visit the 'Existing Characters' page to see who else is playing! On this page, you can perform several tasks. First and foremost, you are able to see the characters of the other users. You are also able to search through these users by name if you use the search bar on the right hand corner of the screen. From this list, you will also be able to edit existing characters.

View/Modify a character

You can view/modify a character by clicking on the pencil icon in their row. A modal will pop up and provide you with further instructions from there.

Delete a character

Similarly, you can delete a character by click on the 'X' icon in the character's row. A modal will pop up and provide you with further assistance.
